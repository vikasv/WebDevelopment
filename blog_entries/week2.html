Week3 : CSS

During the third week of the course, my experiments were around exploring CSS and CSS3. Having fundamentals of HTML will get you started on making a web page. But the default styling of components on the web page is pretty basic and CSS i.e, Styling sheet is must to add the aesthetic feel to your web page. Hence, I have started learning CSS and CSS3 to make my blog much more appealing and attractive. Learning CSS takes opens a new window to the world of Web Designing and there is a lot more you can do on top of a HTML page. I was amazed to see the difference brought by adding CSS to my page and its a great experience playing around with CSS. I have listed below the experiments I have done during the week:

  1. Hover effect: We generally click on a button or link to perform an action. But there are some actions that could be done before actually clicking around. Those can be performed using Hover effect. Image hovering
  2. Border-image property: Any data or a component like label or a table can have a border as lines or dotted lines. But to have an image as a border for a text, we use border-image property to achieve this. Sample code: 
                    #border
                        {
                            border-image: url(border.png) 20 20 round;
                        }

    Rotation effect: We can have a paragraph or any component inside a "div" and have the "div" rotated at a given angle. 
                    Sample code: 
    
                    twoD
                    {
                        margin:20px;
                        padding-top:20px;
                        height:200px;
                        width:100px;
                        background-color:red;
                        transform:rotate(40deg);
                    }

    Rotate when clicked: CSS3 allows features like rotating a "div" when clicked. Before CSS3, it was done using JavaScript but now there is no need of that. CSS3 have many more exhaustive features like this and provides a thrilling experience while exploring number of amazing tricks and hacks. Image effects
  3. Fixed background image:
    In this page, the background image is fixed by using fixed property and also used hover effect. Fixed background
  4. Email Id validation: I have also experimented on validating an email address using bootstrap. Email Id validation
