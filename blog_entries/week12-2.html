Fetching movies statistics

Here, I am using RottenTomatoes API. I have implemented 4 features based on the data I fetch from the API. They are:

  • Display movie currently in theatres which includes a lot of information about each movie, say, date of release, synopsis of the movie, cast, length, etc.
  • Showcase highest grossing movies at the Box Office so far in the descending order.
  • Clicking on the movie poster takes you to IMDB page if one would like to see the ratings and reviews of the movie.
  • Movies releasing this weekend
  • Future releases

Every detail is sent as an AJAX request and received as JSONP response through Rotten Tomatoes API .
