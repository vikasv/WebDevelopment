  2. Image animation using Jumbotron

    A jumbotron indicates a big box for calling extra attention to some special content or information. A jumbotron is displayed as a grey box with rounded corners. It also enlarges the font sizes of the text inside it. Use a 'div' element with class .jumbotron to create a jumbotron.

    Here, I have used effects on x-axis inorder to move the banner from right to left 

                          banner.css('backgroundPosition', x + 'px' + ' ' + y + 'px');
                      window.setInterval(function () {
                            banner.css("backgroundPosition", x + 'px' + ' ' + y + 'px');
                            x--;
                      }, 90);
