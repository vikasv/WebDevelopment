Important tags

P tag:

To write something as paragraph's we use p tags. We cant write text as paragraph without using this tag as white spaces are ignored by html.

del tag:

To show some content as striked we use this tag

b tag:

To show some content as bold we use this tag

strong tag:

To highlight some text we use this tag

i tag:

Content is in italic font

em tag:

To emphasize some content we use this tag

small tag:

To reduce the size of text, we use small tag

mark tag:

To mark some text we use this tag

sub tag:

Main text Text is written as subscript

sup tag:

Main text Text is written as superscript

pre tag: 

         Text in a pre element is displayed in a fixed-width font (usually Courier),
